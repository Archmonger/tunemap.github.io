About Us

TuneMap was built by a team of students from the Computer Science Department at California State University Fullerton. We decided to implement this project as an exercise in learning how to interact with external frameworks, RESTful APIs, and working as part of a team. These are some of the languages and technologies used to create TuneMap:

  • HTML5, CSS3, ES6
  • json-server
  • Node.js
  • Node Package Manager
  • Handlebars.js
  • jQuery
  • justifiedGallery
Made by the TuneMap Team (2019)
